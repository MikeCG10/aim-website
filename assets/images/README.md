# Images

This folder is a placeholder. The site currently uses text/CSS placeholders instead of real images (event photos, officer headshots, club logo) so it renders without any external assets.

To swap in real content:
- Add event photos here and replace the `.card-image-placeholder` divs in `events.html` with `<img>` tags pointing to your files.
- Add officer headshots here and replace the `.avatar` divs in `team.html` with `<img>` tags.
- Replace `assets/logo.svg` with your real club logo (keep the filename or update the `<img src>` references in the page headers).
